# Wave-11 预注册假设清单：TA 变种/联动（2026-07-07，跑数前写死）

**纪律**：每个假设先写"为什么该有效"+ 可证伪预测 + 死刑标准，全跑 2016-2026 长窗，
family=ta_combo 统一记账。过 Bonferroni（~2.9）者进三关卡（size/low_vol/high_52w
partial IC）+ 栖息地诊断。禁止事后在网格里挑格子。

## 动物园验尸的核心线索（假设的出发点）

八个死指标不是随机死的——**四个反转味指标符号全部一致向反转**
（bollinger_b +2.55、mfi +1.9、rsi +1.4、macd_hist -2.95 即反着趋势 = 反转），
t 值都卡在 1.4-2.9 之间。这个模式与"根本没有信号"不同：更像**存在一个真反转
潜因子，每个指标都是它的高噪声测量**。这直接催生 H1；H2/H3 从机制上问
"反转何时才该有效"；H4 处理趋势族死因里的波动混淆（顺带覆盖用户点名的超级趋势）。

## H1 反转集成（reversal_ensemble）——方差缩减

- **逻辑**：四个独立构造的反转测量（通道位置/量加权超买/价格超买/趋势加速度反号）
  秩平均，测量噪声 ~1/√4 缩减。若潜因子为真，集成 t 应显著高于任何单指标。
- **预测**：IC t > 2.9（单指标最好 2.55）；且过三关卡后仍有残差（短反转裸版已死
  t=1.94，集成若只是它的马甲会被斩——那也算干净结案）。
- **死刑**：t < 2.55（连最好单指标都不如 = 无共同潜因子）。

## H2 放量条件化反转（volume_confirmed_reversal）——wave-3 机制迁移

- **逻辑**：反转收益是流动性提供的报酬——无量的价格位移没人被迫接盘，无溢价可赚。
  wave-3 已在日内尺度证明该机制（尾盘量能条件化让 IC 全 horizon 单调增强）。
  日频版：bollinger_b × 异常成交量（21 日均量的倍数，capped 3）。
- **预测**：条件化版在全部 4 个 horizon 上 ≥ 裸 bollinger_b（与 wave-3 同构的单调增强）。
- **死刑**：任一 horizon 反而变弱 = 机制不迁移，记录后关闭。

## H3 锚定条件化反转（anchor_gated_reversal）——双排序

- **逻辑**：George-Hwang 锚定——贴近 52 周高的位移是信息（对好消息反应不足→漂移），
  远离高点的位移是噪声/流动性（税盾卖压、恐慌）→ 反转集中在**远离锚**的名字。
- **预测**：低 high_52w 半区内的反转 IC 显著 > 高半区（区间内差异 t 检验）。
- **死刑**：两半区无差异 = 锚定不调制反转。

## H4 ATR 归一化趋势（supertrend 族）——波动混淆排除

- **逻辑**：sma_gap 死可能死于波动混淆——原始偏离度的横截面排名被高波动名字
  垄断（同样 10% 偏离对波动 60% 的名字是噪声、对 15% 的名字是强趋势）。
  超级趋势的本质正是 ATR 缩放。测 sma_gap / ATR(21)。
- **预测（诚实的低先验）**：若波动混淆是死因，归一化版显著性上台阶；
  更可能仍死（趋势族横截面口径全灭）——那就把"超级趋势/ATR 族"整族结案。
- **死刑**：|t| < 1.5，连同 supertrend 变体一起进总账死亡行。

## 不做的（防网格挖矿）

- 8×8 全组合交互——无先验逻辑的组合不跑。
- 参数扫描（窗口 14→20→50...）——单一预注册参数，不调参。
- 时序择时（supertrend 的本行）——横截面因子口径外，另立项才做。
